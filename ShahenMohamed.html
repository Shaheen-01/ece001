What is meant by Programing language?

A programming language is a notation designed to connect instructions to a computer or machine. Programming languages are primarily used to monitor a machine 's output, or to express algorithms. There are actually thousands of programming languages to be introduced. Many languages must be specified in an imperative form in the computer field, while other programming languages Using declarative form. The software can be divided into two types such as syntax and semanthetic. Some languages are defined by a standard of SO such as the C language.

A programming language is a vocabulary and set of grammatical rules for instructing a computer or computing device to perform specific tasks. The term programming language usually refers to high-level languages, such as BASIC, C, C++, COBOL, Java, FORTRAN, Ada, and Pascal. Each programming language has a unique set of keywords (words that it understands) and a special syntax for organizing program instructions.
