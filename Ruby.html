Ruby is a scripting language built from the ground up for use in front end and back end web development and similar applications. It is a robust, dynamically typed, object-oriented language with syntax so easy to understand and high-level that it’s considered as close as you can get to coding in English.Released in the 1990s, Ruby is an open-sourced language created by the Japanese programmer Yukihiro “Matz” Matsumoto. He has stated that the language is designed to be both fun and productive. Ruby is scripted, meaning that it’s an interpreted language rather than a compiled one.
